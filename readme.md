# Pagina sobre pokemon quetzal

## Descripcion

Pokemon quetzal es un hack rom de pokemon esmeralda que agrega un modo multijugador, tiene los pokemones de la primera a la novena generacion.

Este hack rom es creado por TenmaRH que tiene un canal de youtube, donde sube las actualizaciones y avances de este juego.

Este proyecto es una pagina de este juego donde se pone informacion sobre este juego

## Caracteristicas

Para esta pagina se usa:
    Tabla de contenido
    Imagenes sobre el juego
    Tabla de informacion
    video de youtube
    cuadros para descargar el juego

## Tecnologias usadas

Para esta pagina se uso un html y un css pero tambien se uso los ejemplos de bootstrap para hacer las tablas y mas.
